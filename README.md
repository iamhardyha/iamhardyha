### 🚀 핵심 역량

- **대규모 트래픽 백엔드 설계·운영** — 누적 900만 사용자 / 월 MAU 100만 / 일 30만 건 트래픽 환경 운영
- **이벤트 시스템 자동화** — 코드에 흩어진 이벤트 로직을 DB 스키마 기반으로 재설계, 약 50개 이벤트 API 안정화
- **제휴 플랫폼 구축·확장** — 금융·미디어 등 약 20개 제휴사 연동 주도, 제휴 유형을 공통 구조로 추상화해 신규 온보딩 비용 절감
- **신사업 0 → 1** — 요구사항 분석 → DB 모델링 → API 설계·운영까지 서버 전반 주도
- **운영 환경 고도화** — JDK 8 → JDK 21, MySQL → Aurora MySQL 3 마이그레이션 수행
- **데이터 · 협업** — 대규모 데이터 조회/가공 API 설계, 쿼리 최적화, 비개발 직군과의 요구사항 정리
